## Python ATM Banking System (School Project)

This project is a simple **terminal-based ATM banking system** built using **Python and basic Object-Oriented Programming (OOP)** concepts. The program simulates how an ATM machine works, allowing users to create accounts and perform basic banking transactions.

User accounts are stored permanently using **JSON files**, allowing the system to save and load account data even after the program is closed.

The program also includes **terminal animation effects**, such as typing text and loading screens, to make the experience more interactive and engaging.

### Features

* Create a new bank account
* Secure login using Account ID and PIN
* Deposit money
* Withdraw money with maintaining balance rule
* Check current balance
* View transaction history
* Persistent data storage using JSON
* Terminal typing and loading effects for a better user experience

### Concepts Used

* Object-Oriented Programming (Classes and Methods)
* File handling
* JSON data storage
* Loops and conditional statements
* Error handling
* Terminal interface design

This project was created as part of a **Python school project** to practice programming fundamentals and simulate a real-world banking system.
