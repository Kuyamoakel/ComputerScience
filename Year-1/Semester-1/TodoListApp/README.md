Python To-Do List (Basic CRUD Project)

This is a simple command-line To-Do List application built with Python. The program allows users to manage their daily tasks by performing basic CRUD operations (Create, Read, Update, Delete).

Tasks are stored using a JSON file, which allows the data to persist even after the program is closed. When the program starts, it automatically loads previously saved tasks, and every change updates the JSON file.

This project was created as a personal practice project to improve my understanding of Python fundamentals such as file handling, functions, lists, and program flow.

Features

Add new tasks

View all saved tasks

Remove tasks

Automatic saving using JSON

Persistent storage (tasks remain saved after closing the program)

Concepts Used

CRUD Operations (Create, Read, Delete)

Python Lists

Functions and Program Flow

JSON File Handling

Error Handling (FileNotFoundError and JSONDecodeError)

Command-line Interface (CLI)

This project helped strengthen my understanding of basic Python programming and data persistence using JSON files.
